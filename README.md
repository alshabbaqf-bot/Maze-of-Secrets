# Maze of Secrets

Maze of Secrets is a simple 3D maze game built using Unity.

## Game Idea
The player navigates through a maze to collect coins and reach the exit before time runs out.

## Features
- Player movement using keyboard
- Collectible coin system (score tracking)
- Countdown timer
- Win and lose conditions
- Main menu (Play / Quit)
- Result screen with restart and main menu options

## How to Play
- Use **WASD** or **Arrow Keys** to move
- Collect all coins
- Reach the exit to win
- Finish before the timer reaches zero

## Controls
- W / A / S / D → Move
- Arrow Keys → Move
